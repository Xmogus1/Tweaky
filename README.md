# Tweaky

A client-side Fabric mod for Minecraft **26.1.2** and **26.2** - QoL tweaks and visual upgrades.

**[⬇ Download](../../releases/latest)**

Needs Fabric Loader, Fabric API, and Fabric Language Kotlin.

In game: `/tweaky`
Hud edit `/tweaky hud`
